# TaskFlow – Personal Task Manager

A lightweight browser-based task manager built with Angular v21.

## Student Info
- **Name:** Your Full Name
- **Student ID:** 20242470
- **Course:** Your Course and Year
- **Section:** Your Section

## Features
- Add, edit, delete tasks
- Toggle task status (Pending / In Progress / Completed)
- View all tasks in a list
- View full task details on a separate page
- Priority levels: Low, Medium, High

## Tech Stack
- Angular v21 (Standalone Components)
- TypeScript
- Bootstrap 5
- Angular Router with child routes

## How to Run
```bash
npm install
ng serve
```
